Develop Your Own Car - MbSE at Fortiss

Working in a team of four, we used Model-based systems engineering (MbSE) to develop software for a small, self-driving car model . We implemented steering, braking, gears (and ofc a sport mode), but mainly advanced driver-assistance systems like adaptive cruise control, emergency breaking, and lane-keeping. I learned a lot about agile software development by completing a full development process of modeling, simulating and deploying/testing on hardware from a set of requirements.
Tools used: AutoFOCUS3, OpenModelica, Matlab, Gazebo (simulation), Python (Matplotlib, Numpy), Git

hackaTUM 2023 - Check24 Challenge

During the hackaTUM 2023 I developed an MVP for CHECK24's craftsmen comparison software engineering challenge in a team of four. We implemented a frontend and backend for a geo-distance, weighted ranking algorithm that ranks craftsmen based on their distance to the customer, their rating, and their price. Additionally, we optimized our MVP for speed with pre-calculations on the database. Working with new technologies and a very limited amount of time was a fun and challenging experience which I would love to repeat.
Tools used: SvelteKit, TailwindCSS, Rust, Diesel, PostgreSQL, Docker, Git

Bachelor's Thesis

For my bachelor's thesis, I designed, implemented, and executed a simulation-based benchmark suite for modular reconfigurable robots based on NIST's specifications.
